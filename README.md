### Mesto Express 

Backend Mesto на Node.js (Express).

#### Скрипты:
Для запуска сервера на **localhost:3000** 
`npm run start`

Для запуска с hot reload
`npm run dev`

Проверка ESLint
`npm run lint`

Исправление ошибок ESLint:
`npm run lint:fix`
